# AI Chatbot with Voice & GUI
An AI-powered chatbot with voice recognition and speech synthesis using Google Gemini AI and Tkinter-based GUI.

## Overview
This is an AI-powered chatbot with voice recognition and speech synthesis, built using Google Gemini AI, speech recognition, and a custom Tkinter-based GUI. The chatbot can:
- Respond to user queries using AI-generated responses.
- Recognize voice input and convert it to text.
- Read responses aloud using text-to-speech.
- Perform Google searches, YouTube searches, and play YouTube videos.
- Feature a modern, animated GUI with user and bot avatars.

## Features
- **AI-Powered Chat**: Uses Google's Gemini AI for intelligent responses.
- **Voice Commands**: Supports speech recognition for hands-free interaction.
- **Text-to-Speech**: Converts bot responses into spoken words.
- **Custom GUI**: Built with Tkinter and CustomTkinter for a visually appealing interface.
- **Animated GIF Integration**: Displays an animated bot avatar.
- **Search Capabilities**: Open Google searches and YouTube videos directly from the chat.

## Technologies Used
- `google.generativeai` (Google Gemini AI)
- `speech_recognition` (Voice Input)
- `pyttsx3` (Text-to-Speech)
- `tkinter` & `customtkinter` (GUI)
- `Pillow` (Image Processing)
- `pywhatkit` & `webbrowser` (Web Searches & YouTube Playback)
